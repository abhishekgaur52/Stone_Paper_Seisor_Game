# Stone_Paper_Scissor_Game
```
A simple command-line **Stone Paper Scissors** game written in Python. The player competes against the computer, which randomly selects its move.
```
## Features
```
* Random computer choice
* User input from the terminal
* Displays both player and computer choices
* Announces the winner or a draw
* Beginner-friendly Python project
```
## Technologies Used
```
* Python 3
* `random` module
```

## Game Controls

```
0 = Stone
1 = Paper
2 = Scissors
```

Enter the corresponding number when prompted.

## Example

```
0 = Stone
1 = Paper
2 = Scissors

ENTER YOUR NUMBER: 1

You chose: Paper
Computer chose: Stone

YOU WIN
```
